# 📚 AI Campus Assistant Bot

The AI Campus Assistant Bot is a smart web-based chatbot designed to help students quickly access important campus information such as timetable, exams, faculty details, and events. It reduces manual enquiry work and provides instant responses through a simple chat interface.

## 🚀 Features

- 💬 Simple chatbot interface for students  
- 📅 Provides timetable and exam-related information  
- 🎓 Shows campus events and updates  
- ⚡ Fast responses using rule-based AI  
- 🌐 Built using HTML, CSS, and JavaScript    

---

## 🛠️ Tech Stack

HTML, CSS, JavaScript (Frontend)  
Python (Optional Backend)  
Rule-based-AI

## Conclusion

The AI Campus Assistant Bot successfully demonstrates how a simple rule-based system can improve communication and information access within a campus environment. By using keyword-based logic, it provides quick and accurate responses for common student queries such as timetables, exams, faculty details, and events.

This project reduces the need for manual enquiries and makes campus information easily accessible through a user-friendly chat interface. Although it is currently rule-based, it can be further enhanced by integrating AI models or NLP APIs for smarter and more flexible responses.
