# 📚 AI Campus Assistant Bot

The AI Campus Assistant Bot is a smart web-based chatbot designed to help students quickly access important campus information such as timetable, exams, faculty details, and events. It reduces manual enquiry work and provides instant responses through a simple chat interface.

## 🚀 Features

- 💬 Simple chatbot interface for students  
- 📅 Provides timetable and exam-related information  
- 🎓 Shows campus events and updates  
- ⚡ Fast responses using rule-based AI  
- 🌐 Built using HTML, CSS, and JavaScript    

---

## 🛠️ Tech Stack

HTML, CSS, JavaScript (Frontend)  
Python (Optional Backend)  
Rule-based-AI
